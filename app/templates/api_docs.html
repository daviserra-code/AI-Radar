REST API per accedere ai contenuti di AI-RADAR

GET /api/articles

Ottieni lista di articoli con paginazione

Parametri Query:

  • limit (int, default: 20) - Numero di articoli per pagina
  • offset (int, default: 0) - Offset per paginazione
  • category (string) - Filtra per slug categoria

Esempio:

GET /api/articles?limit=10&category=llm

GET /api/article/{slug}

Ottieni dettagli completi di un articolo

Esempio:

GET /api/article/llama-3-2-release

GET /api/search

Ricerca articoli per parole chiave

Parametri Query:

  • q (string, required) - Testo da cercare
  • limit (int, default: 20) - Numero massimo risultati
  • category (string) - Filtra per categoria

Esempio:

GET /api/search?q=gpu&category=hardware

GET /api/categories

Lista categorie con statistiche

Esempio:

GET /api/categories

GET /api/tags

Lista tag popolari con conteggi

Parametri Query:

  • limit (int, default: 50) - Numero massimo tag

Esempio:

GET /api/tags?limit=20
